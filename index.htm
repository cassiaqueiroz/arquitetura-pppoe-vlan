Arquitetura PPPoE/VLAN

Fluxo completo de autenticação e conexão à internet

6 Etapas ~2-5 segundos Seguro
1

Roteador Cliente

Inicia descoberta PPPoE com credenciais configuradas

PPPoE Discovery PADI
PPPoE Discovery:
O roteador cliente envia um pacote PADI (PPPoE Active Discovery Initiation) para descobrir servidores PPPoE disponíveis na rede.
2

Tagging VLAN

Cada quadro Ethernet recebe uma etiqueta VLAN ID única para isolamento e identificação do serviço específico

VLAN 100 802.1Q
VLAN Tagging (IEEE 802.1Q):
Cada pacote recebe uma "etiqueta" de 4 bytes contendo a VLAN ID (ex: 100). Isso permite que múltiplos clientes compartilhem a mesma infraestrutura física mantendo o tráfego isolado e identificado por serviço/cliente.
3

OLT/PON

Processa tráfego da fibra óptica

GPON Fibra
OLT (Optical Line Terminal):
Equipamento que gerencia múltiplas ONUs na rede PON, concentrando o tráfego de fibra óptica.
4

BRAS/BNG

Servidor PPPoE processa autenticação

PPPoE Server Session
BRAS/BNG:
Broadband Remote Access Server - equipamento que termina sessões PPPoE e gerencia o acesso dos assinantes.
5

Servidor RADIUS

Valida credenciais e políticas

AAA Authentication
RADIUS Server:
Sistema de autenticação, autorização e contabilização que verifica credenciais, status de pagamento e define políticas de acesso.
6

Conexão Ativa

Sessão PPPoE estabelecida com sucesso

Online
Conexão Estabelecida:
RADIUS retorna Access-Accept com configurações de IP, velocidade e políticas. Cliente agora tem acesso à internet.

Glossário

Definições detalhadas dos componentes da arquitetura PPPoE/VLAN

OLT

Optical Line Terminal

Definição: Equipamento central que gerencia múltiplas ONUs em uma rede PON.

Função: Concentra o tráfego de fibra óptica e realiza conversão óptico-elétrica.

Localização: Central do provedor ou ponto de distribuição.

Capacidade: até 128 ONUs Alcance: até 20km

PON

Passive Optical Network

Definição: Rede de fibra óptica passiva que compartilha uma única fibra entre múltiplos usuários.

Função: Permite distribuição eficiente de banda larga via fibra óptica.

Vantagem: Reduz custos de infraestrutura e manutenção.

Topologia: Ponto-Multiponto Divisão: 1:32 ou 1:64

VLAN

Virtual Local Area Network

Definição: Tecnologia que adiciona uma "etiqueta" de 4 bytes aos quadros Ethernet, criando redes virtuais isoladas em uma mesma infraestrutura física.

Função: Permite que múltiplos clientes/serviços compartilhem equipamentos mantendo tráfego totalmente isolado e identificado.

Padrão: IEEE 802.1Q - cada VLAN ID funciona como uma "etiqueta de identificação" única.

IDs: 1-4094 Overhead: 4 bytes

RADIUS

Remote Authentication Dial-In User Service

Definição: Protocolo de rede para autenticação, autorização e contabilização centralizadas.

Função: Valida credenciais, define políticas e monitora uso de recursos.

Aplicação: AAA (Authentication, Authorization, Accounting).

Porta: 1812/1813 Protocolo: UDP

ISP

Internet Service Provider

Definição: Empresa que fornece acesso à internet e serviços relacionados.

Função: Conecta usuários finais à internet global através de sua infraestrutura.

Serviços: Banda larga, hospedagem, email, DNS.

Tier 1/2/3 BGP/MPLS

PPPoE

Point-to-Point Protocol over Ethernet

Definição: Protocolo que encapsula PPP em quadros Ethernet.

Função: Permite autenticação e estabelecimento de sessões IP sobre Ethernet.

Processo: Discovery → Offer → Request → Session.

RFC 2516 MTU: 1492 bytes

GPON

Gigabit Passive Optical Network

Definição: Tecnologia PON que oferece velocidades de até 2.5 Gbps downstream.

Função: Transmissão de dados, voz e vídeo em alta velocidade via fibra.

Padrão: ITU-T G.984 para redes de acesso ópticas.

Down: 2.488 Gbps Up: 1.244 Gbps

Informações Técnicas

PPPoE

Point-to-Point Protocol over Ethernet. Protocolo que permite autenticação e estabelecimento de sessões IP sobre redes Ethernet.

VLAN

Virtual Local Area Network. Segmentação lógica de rede que permite isolamento de tráfego em uma mesma infraestrutura física.

PON/GPON

Passive/Gigabit Passive Optical Network. Tecnologia de fibra óptica que permite compartilhamento de uma única fibra entre múltiplos usuários.

RADIUS

Remote Authentication Dial-In User Service. Protocolo de rede que fornece autenticação, autorização e contabilização centralizadas.
